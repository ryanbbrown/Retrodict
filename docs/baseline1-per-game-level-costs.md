# baseline1 per-game level economics

Same shape as [per-game-level-costs.md](per-game-level-costs.md), computed from [baseline1](https://github.com/astroseger/arc-3-agents-baseline1)'s published archive (`secure_baseline1_v1.5_gpt5.5_xhigh_run01`, gpt-5.5 at xhigh reasoning effort), one run per game. Generated by `scripts/gen_baseline1_level_costs.py` from the archive's per-game scorecard and cost-estimation JSONs; the script header documents how to download and extract them.

Actions split between **cleared levels** and the **uncleared level** using the per-level action counts in each game's scorecard. The archive has no per-invocation usage logs, so **tokens and cost are whole-game totals** from baseline1's cost-estimation files. Token counts are taken directly from those files (tokens = input + output, cache reads included in input). Cost is baseline1's own `estimated_cost_usd`, computed at GPT-5.5 API list prices ($5.00 input / $0.50 cached / $30.00 output per MTok — the same rates as gpt-5.6-sol, so the columns are arithmetically comparable). baseline1 ran through Codex, so this is an API-price equivalent, not what the run actually cost to whoever ran it; **tokens are the comparable measure**, cost is shown for scale.
