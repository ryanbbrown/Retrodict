# baseline1 per-game costs

Same shape as [per-game-costs.md](per-game-costs.md), for [baseline1](https://github.com/astroseger/arc-3-agents-baseline1)'s two gpt-5.6-sol max-effort runs, one table each: twma (textual world model agent) and ewma_sv (executable world model agent with verification, its strongest configuration). Generated by `scripts/gen_baseline1_game_costs.py` from the per-game scorecard and cost-estimation JSONs baseline1 published with its paper02 run archives.

Score, levels, and actions come from each game's server scorecard; token counts are baseline1's own raw usage numbers (input includes cache reads, shown separately in the cached column). Cost is recomputed at API list prices ($5.00 input / $0.50 cached / $30.00 output per MTok, the same rates as my runs, so the columns are arithmetically comparable) and verified against baseline1's published estimates. baseline1 ran through Codex, so this is an API-price equivalent, not what the run actually cost to whoever ran it; **tokens are the comparable measure**, cost is shown for scale.
